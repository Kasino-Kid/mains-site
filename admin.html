Guide d'utilisation de l'admin
📋 Fonctionnement général

Cet admin modifie le fichier contenu.json qui se trouve dans ton dossier. Le site lit ce fichier à chaque chargement pour afficher le bon contenu.


Étapes pour modifier le site :

  1. Modifie ce que tu veux dans l'admin
  2. Clique sur "Enregistrer" en bas de la page
  3. Un fichier contenu.json est téléchargé automatiquement
  4. Remplace l'ancien contenu.json dans ton dossier par le nouveau
  5. Rafraîchis le site — les changements apparaissent !

⚠️ Important

Tous ces fichiers doivent toujours être dans le même dossier :

index.html · admin.html · contenu.json · logo.png · hero.jpg · photo1.jpg ... etc.
